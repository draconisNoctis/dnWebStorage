dnWebStorage
===

Develop
---

### Requirements

 - Grunt-Cli
 - Bower

### Install

 - clone/download repository
 - `npm install`

### Commands

 - `grunt build`  
   builds the dist files
 - `grunt tests`  
   builds and executes tests via phantomjs, open _SpecRunner.html to run tests in browser
 - `grunt coverage`  
   creates a code coverage report
 - `grunt`  
   builds dist files and start watcher
